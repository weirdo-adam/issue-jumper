# Code of Conduct

Issue Jumper welcomes respectful, constructive participation.

Participants are expected to:

- Be professional and considerate.
- Keep discussions focused on the project.
- Respect differing technical opinions.
- Avoid harassment, discrimination, personal attacks, and disruptive behavior.
- Avoid sharing private information without permission.

Maintainers may edit, hide, or remove comments and contributions that violate these expectations. Repeated or severe violations may result in being blocked from participating in the project.

Report conduct issues privately through the repository owner's GitHub profile.
